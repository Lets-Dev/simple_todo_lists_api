Todo List Items API

Editing a to-do list item

PUT todo_list_items/:id

Parameters

Name Description
todo_list_item[label] Label of the item
todo_list_item[status] Status of the item (pending or done)
todo_list_item[todo_list_id] ID of the parent to-do list

Request

Headers

Accept: application/json
Content-Type: application/json
Host: example.org
Cookie:

Route

PUT todo_list_items/1

Body

{"todo_list_item":{"label":"Mustafar","status":"done","todo_list_id":2}}
